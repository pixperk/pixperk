#### the tier list

<div>
  <img src="./tier-list.svg" width="100%" alt="project tier list" />

<details>
<summary>&nbsp;<b>S tier</b></summary>
<br>

&nbsp;&nbsp;[**juzfs**](https://github.com/pixperk/juzfs) · google file system paper in rust. master server, chunk replication, lease-based mutations, namespace management. the whole thing.<br>
&nbsp;&nbsp;[**cineyma**](https://github.com/pixperk/cineyma) · erlang-style actor framework for rust. supervision trees, typed messaging, failure isolation, remote actors, clustering.

</details>

<details>
<summary>&nbsp;<b>A tier</b></summary>
<br>

&nbsp;&nbsp;[**plethora**](https://github.com/pixperk/plethora) · amazon dynamo paper. consistent hashing, vector clocks, gossip protocol, sloppy quorum.<br>
&nbsp;&nbsp;[**pixtorrent**](https://github.com/pixperk/pixtorrent) · bittorrent from the wire up. tracker protocol, peer swarm, piece selection, concurrent downloads.<br>
&nbsp;&nbsp;[**redis_in_rust**](https://github.com/pixperk/redis_in_rust) · full async redis server. RESP from scratch, RDB/AOF persistence, pub/sub, transactions.<br>
&nbsp;&nbsp;[**lowkey**](https://github.com/pixperk/lowkey) · distributed lock service on raft. fencing tokens, lease-based locks, CP-consistent.<br>
&nbsp;&nbsp;[**dokidb**](https://github.com/pixperk/dokidb) · LSM-tree storage engine. memtable, WAL with CRC32, SSTable compaction, background merge.<br>
&nbsp;&nbsp;[**neu**](https://github.com/pixperk/neu) · RFC 6455 websocket server in c++. raw sockets, epoll, openssl. no libraries, no mercy.<br>
&nbsp;&nbsp;[**chug**](https://github.com/pixperk/chug) · streaming ETL pipeline. postgres → clickhouse with constant memory, real-time CDC, web UI.

</details>

<details>
<summary>&nbsp;<b>B tier</b></summary>
<br>

&nbsp;&nbsp;[**mprdc**](https://github.com/pixperk/mprdc) · google's mapreduce (2004) paper implementation in go.<br>
&nbsp;&nbsp;[**vaultify**](https://github.com/pixperk/vaultify) · secrets management. AES-256, version history, audit logs, RBAC, PASETO auth.<br>
&nbsp;&nbsp;[**grit**](https://github.com/pixperk/grit) · git-style version control for spotify & youtube playlists. branch, merge, diff + TUI player.<br>
&nbsp;&nbsp;[**nosynK**](https://github.com/pixperk/nosynK) · async job queue system in go.<br>
&nbsp;&nbsp;[**uptimex**](https://github.com/pixperk/uptimex-server) · real-time server & SSL monitoring. graphql subscriptions, websocket alerts.

</details>
